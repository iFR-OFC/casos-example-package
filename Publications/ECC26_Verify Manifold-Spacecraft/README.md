This repository contains the research code accompanying the paper *Sum-of-Squares Stability Verification on Manifolds with Applications in Spacecraft Attitude Control*.

# Requirements
- [Matlab](https://de.mathworks.com/products/matlab.html)
- [Matlab Symbolic Math Toolbox](https://de.mathworks.com/products/symbolic.html)
- [Git](https://git-scm.com/)
- [CasADi 3.6 or newer](https://web.casadi.org/get/)
- [Mosek 11 (older versions could work aswell)](https://docs.mosek.com/11.0/install/installation.html)

# Installation

1. Open a terminal and run
```bash
git submodule update --init --recursive
```
in the root folder of this repository.

2. In Matlab, double click the project file to initialize the path/environment.
